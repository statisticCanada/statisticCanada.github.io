Concept Model

The data is created based on approximations and is for visualization purpose only.


User guidelines:

Visualization:
 1. The click on categories above the visualization will direct the you to the data table and relationship 
    graph for respective category. 
 2. The Concept map gives the relation between all the categories and subcategories based on thematic model.  
 3. The blue and red lines are called links or paths.  
 4. The rectangular boxes are called nodes.  
 5. It is a hierarchical graph where nodes can expand and collapse.  
 6. When you double clicks on nodes, the node will collapse.  
 7. The collapsed node will appear on the top right of the visualization.  
 8. The red paths are outflows to the node and blue are inflows.  
 9. The nodes can be dragged dynamically to any part of the page to visualize better.  
10. If you click on paths, you will be directed to Statistics Canada’s website portal
    which will include a list of all the data set belonging to that node.

Table:
 1. The click on categories above the visualization will direct you to the data table and relationship 
    graph for respective category. 
 2. There are five different categories which appear as a button on top of the page.
 3. Below each category there are subcategories associated to it.
 4. You can click on both category and subcategory.
 5. Click on category will direct you to the category model view.
 6. Click on subcategory will direct you to the data tab containing the list of data set
    belonging to respective subcategory.
 7. For further documentation refer out github page.
